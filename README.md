# エレクトロニクスⅠ 視覚学習サイト

電気・電子工学の基礎用語を、**図解・アニメーション・比較表**で直感的に理解できる学習サイトです。

## 特徴

- **49の重要用語**を網羅（電子・電荷、抵抗・回路、磁気、交流、法則）
- 各用語ごとに以下を提供：
  - 一言でわかる定義
  - SVG による視覚的な図解
  - 原理の説明（「なぜそうなるのか」を重視）
  - よくある間違い
  - 関連用語との違い
- **比較表**で混同しやすい用語を整理
- カテゴリ別フィルタ・検索機能
- ダークモード対応
- モバイル / PC 両対応

## 使い方

1. カテゴリボタンで絞り込み、または検索バーで用語を検索
2. 用語カードをクリックして詳細パネルを表示
3. 「比較表を見る」ボタンで、似ている用語を比較
4. ハッシュリンク（`#用語ID`）で直接アクセス可能

## 技術スタック

- バニラ JavaScript（フレームワークなし）
- CSS カスタムプロパティ（ダークモード）
- SVG によるインライン図解
- ローカルストレージでテーマ保存
