## 📚 収録内容

### カテゴリ別用語一覧

#### 🔋 電子・電荷
電子、自由電子、電荷、クーロンの法則、電流、電圧、抵抗、導体と不導体、オームの法則

#### 🔌 抵抗・回路
直列回路、並列回路、キルヒホッフの法則、電力と電力量、ジュール熱、電圧降下、起電力、内部抵抗

#### 🧲 磁気
磁力線、磁界、磁束、磁束密度、透磁率、右ねじの法則、フレミング左手の法則、フレミング右手の法則、電磁力、レンツの法則、相互誘導、自己誘導

#### 🌊 交流
交流と直流、周波数と周期、実効値、位相と位相差、誘導性リアクタンス、容量性リアクタンス、インピーダンス、力率、有効電力と無効電力

#### ⚖️ 法則
クーロンの法則、オームの法則、キルヒホッフの法則、ジュール熱の法則、フレミングの左手・右手の法則、レンツの法則

### 📊 比較表

1. **電圧・電位・電位差・電圧降下・起電力** — 単位は同じ[V]だけど意味が違う
2. **電子 vs 自由電子** — どちらも電子だけど「動けるか」が重要
3. **磁界の強さH・磁束Φ・磁束密度B** — 磁気の3大要素を整理
4. **誘導性リアクタンス X_L vs 容量性リアクタンス X_C** — コイルとコンデンサの違い
5. **直列回路 vs 並列回路** — 電流・電圧・合成抵抗の違い

## 🎯 使い方

### 基本操作
1. **カテゴリを選択** して用語を絞り込む
2. **検索ボックス** で用語を直接検索
3. **用語をクリック** して詳細表示
4. **← → ボタン** で前後の用語へ移動
5. **⭐ボタン** でお気に入りに追加

### キーボードショートカット
- `←` `→` : 前後の用語へ移動
- `/` または `Ctrl+K` : 検索フォーカス
- `Esc` : 検索解除・パネル閉じる
- `B` : お気に入りに追加/削除
- `?` : ショートカット一覧表示

### 学習機能
- **進捗バー**: 画面左下に学習進捗を表示
- **閲覧済みマーク**: 見た用語にチェックマーク表示
- **お気に入り**: 重要な用語をブックマークして後で復習
